Mp4Autotag

Beta Release 1.3

This beta release introduces a new workflow for the user interface.

Changes

Now multiple files can be selected for Removing, Saving, Autotagging, and Automating.

  • Removing
    Removing will remove the selected file(s) from the queue.
  • Select file(s) to remove.
  • Click Remove button.
  • Saving
    Saving will save the selected file(s) edited tag data.
    1. Select file(s) to save.
    2. Click Save button.
  • Autotagging
    Autotagging will bring up a search results window for all files selected. It will take a file name, use its full path to decide if the files is a Show or a Movie, then based on that pattern match, search the appropriate databases for information about that show/movie.The search results window will present the user with a list of tags to attach to the file.
    1. Select file(s) to Autotag.
    2. Click Save button.
  • Automated
    Automated will do a batch autotagging of the selected file(s). Each file selected will be searched and the top result will be used to tag the current file. After the process the user is presented a results window where they can see what has been changed, and undo the changes if they are not correct.
    1. Select file(s) to Automated.
    2. Click Automated button.

Since the we added multiple selections, it change the way the work flow felt, so the above changes to the tool bar made it so we no longer needed the confusing Fully Automated option in preferences.

Optimize file now uses mp4v2 2.0.0 MP4Optimize function instead of fixMOOV.
